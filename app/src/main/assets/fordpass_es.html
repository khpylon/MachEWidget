El año pasado, Ford inició la suspensión de las cuentas FordPass para los usuarios que acceden a los datos de su vehículo a través de aplicaciones y/o sitios web de terceros no autorizados. Algunos usuarios de esta aplicación han informado de este problema, aunque no hemos podido confirmar que fue la aplicación la que provocó la suspensión (muchos de los usuarios de esta aplicación también han utilizado otras aplicaciones/sitios web de terceros, como Recurrent). Una discusión en curso sobre estos temas está disponible en los foros de Mach-E.

Cuando se suspende una cuenta, debes comunicarte con FordPass para restablecerla. Le pedirán que deshabilite o elimine todas las cuentas de terceros no autorizados antes de restablecer su cuenta y le advertirán que la próxima infracción puede resultar en una suspensión permanente.

Llevamos más de un año intentando convertirnos en una aplicación autorizada y, aunque todavía no ha sucedido, hemos logrado algunos avances. Mientras tanto, si deseas seguir usando la aplicación te hacemos estas recomendaciones:

  1. Cree una cuenta de correo electrónico FordPass alternativa para utilizar esta aplicación. Las instrucciones para hacerlo se pueden encontrar en https://www.ford.com/support/how-tos/search/How%20Do%20I%20Add%20Another%20User%20to%20My%20Connected%20Vehicle/ .
  2. Incrementar el tiempo entre actualizaciones.
  3. Evite el uso de funciones "remotas" de la aplicación, como bloquear/desbloquear e iniciar/detener. Hazlo a través de la aplicación FordPass.
  4. Asegúrese de que la actualización forzada y las actualizaciones forzadas automáticas estén deshabilitadas en Configuración.

Si desea conservar la aplicación pero no se siente cómodo usándola hasta que recibamos la autorización, habilite "Hibernar la aplicación hasta que se lance la API pública" en Configuración. Esto hará que la aplicación deje de actualizar datos automáticamente hasta que se implemente la API pública. Las actualizaciones manuales desde el menú de tres puntos seguirán funcionando. Aparecerá una notificación en el futuro para informarle que todo está bien.
