No ano passado, a Ford iniciou a suspensão de contas FordPass para utilizadores que acedam aos dados dos seus veículos através de aplicações e/ou websites de terceiros não autorizados. Alguns usuários deste aplicativo relataram esse problema, embora não tenhamos conseguido confirmar se foi o aplicativo que causou a suspensão (muitos dos usuários deste aplicativo também usaram outros aplicativos/sites de terceiros, como o Recurrent). Uma discussão contínua sobre essas questões está disponível nos fóruns do Mach-E.

Quando uma conta é suspensa, você precisa entrar em contato com a FordPass para restabelecê-la. Eles exigirão que você desative ou remova todas as contas de terceiros não autorizadas antes de restabelecer sua conta e avisarão que a próxima violação poderá resultar em uma suspensão permanente.

Há mais de um ano que tentamos nos tornar um aplicativo autorizado e, embora isso ainda não tenha acontecido, fizemos alguns progressos. Enquanto isso, se você quiser continuar usando o aplicativo, fazemos estas recomendações:

  1. Crie uma conta de e-mail FordPass alternativa para este aplicativo usar. Instruções para fazer isso podem ser encontradas em https://www.ford.com/support/how-tos/search/How%20Do%20I%20Add%20Another%20User%20to%20My%20Connected%20Vehicle/ .
  2. Aumente o tempo entre as atualizações.
  3. Evite usar os recursos "remotos" do aplicativo, como bloquear/desbloquear e iniciar/parar. Faça isso por meio do aplicativo FordPass.
  4. Certifique-se de que a atualização forçada e as atualizações forçadas automáticas estejam desativadas em Configurações.

Se você quiser manter o aplicativo, mas não se sentir confortável em usá-lo até recebermos autorização, ative "Hibernar aplicativo até que a API pública seja lançada" em Configurações. Isso fará com que o aplicativo pare de atualizar automaticamente os dados até que a API pública seja implementada. As atualizações manuais no menu de três pontos ainda funcionarão. Uma notificação aparecerá no futuro para que você saiba que está tudo bem.
