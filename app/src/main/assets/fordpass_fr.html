Au cours des derniers mois, Ford a commencé à suspendre les comptes FordPass des utilisateurs qui accèdent aux données de leur véhicule par le biais d'applications et/ou de sites web tiers non autorisés. Certains utilisateurs de cette application ont signalé ce problème, bien que nous n'ayons pas été en mesure de confirmer que c'était l'application qui était à l'origine de la suspension (de nombreux utilisateurs de cette application ont également utilisé d'autres applications/sites web tiers tels que Recurrent). Une discussion en cours sur ce problème est disponible sur les forums de Mach-E.

Lorsqu'un compte est suspendu, vous devez contacter FordPass pour le rétablir. Ils vous demanderont de désactiver ou de supprimer tous les comptes de tiers non autorisés avant de rétablir votre compte, et vous préviendront que la prochaine violation peut entraîner une suspension permanente.

Nous essayons de devenir une application autorisée depuis quelques mois et, bien que ce ne soit pas encore le cas, nous avons fait quelques progrès. En attendant, si vous souhaitez continuer à utiliser l'application, nous vous faisons les recommandations suivantes :

  1. Créez un autre compte courriel FordPass pour que cette application puisse être utilisée. Vous trouverez des instructions à ce sujet sur le site Comment puis-je autoriser un autre utilisateur dans l'application FordPass? .
  2. Augmentez le temps entre les mises à jour.
  3. Évitez d'utiliser les fonctions de l'application « à distance » telles que le verrouillage/déverrouillage et le démarrage/arrêt. Utilisez plutôt l'application FordPass.
  4. Assurez-vous que l'actualisation forcée et les mises à jour automatiques forcées sont désactivées dans les paramètres

Si vous souhaitez conserver l'application mais que vous ne vous sentez pas à l'aise pour l'utiliser jusqu'à ce que nous recevions l'autorisation, activez « Hiberner l'application jusqu'à ce que l'API publique soit publiée » dans les paramètres. L'application cessera alors d'actualiser automatiquement les données jusqu'à ce que l'API publique soit mise en œuvre. Les actualisations manuelles à partir du menu à trois points continueront de fonctionner. Une notification s'affichera à l'avenir pour vous informer que tout va bien.
