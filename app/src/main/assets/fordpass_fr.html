Au cours des derniers mois, Ford a initié la suspension des comptes FordPass pour les utilisateurs qui accèdent aux données de leur véhicule via des applications tierces non autorisées et/ou sites Internet. Certains utilisateurs de cette application ont signalé ce problème, même si nous je n'ai pas été en mesure de confirmer que c'était l'application qui avait provoqué la suspension (De nombreux utilisateurs de cette application utilisent également d'autres applications/sites Web tiers tels que comme récurrent). Une discussion en cours sur ces problèmes est disponible sur les forums Mach-E.

Lorsqu'un compte est suspendu, vous devez contacter FordPass pour avoir il a rétabli. Ils vous demanderont de désactiver ou de supprimer tous les comptes tiers avant de rétablir votre compte, et vous avertir que le la prochaine infraction peut entraîner une suspension permanente.

Nous essayons de devenir une application autorisée depuis quelques mois et bien que cela ne s'est pas encore produit, nous avons fait des progrès. En attendant, si vous souhaitez continuer à utiliser l'application, nous vous faisons ces recommandations:

  1. Créez un autre compte de messagerie FordPass que cette application pourra utiliser. Les instructions pour ce faire peuvent être trouvées sur https://www.ford.com/support/how-tos/search/How%20Do%20I%20Add%20Another%20User%20to%20My%20Connected%20Vehicle/ .
  2. Augmenter le temps entre les mises à jour.
  3. Évitez d'utiliser les fonctionnalités de l'application «à distance» telles que le verrouillage/déverrouillage et le démarrage/arrêt. Faites-les plutôt via l'application FordPass.
  4. Assurez-vous que l'actualisation forcée et les mises à jour forcées automatiques sont désactivées sous «Paramètres».

Si vous souhaitez conserver l'application mais que vous ne vous sentez pas à l'aise de l'utiliser jusqu'à ce que nous recevions l'autorisation, activez «Mettre l'application en veille prolongée jusqu'à ce que l'API publique soit publiée» dans Paramètres. Cela entraînera l'arrêt automatique de l'application actualiser les données jusqu'à ce que l'API publique soit implémentée. Les actualisations manuelles à partir du menu à trois points fonctionneront toujours. Une notification apparaîtra à l'avenir pour vous informer que tout va bien.
